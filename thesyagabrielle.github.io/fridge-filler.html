    • Fridge Filler Mobile App

    Use Case

    In a busy life, shopping lists often become less important, leading to food wastage. People tend to forget updating their shopping lists when they run out of ingredients. Conversely, they may overbuy items they've already purchased, further contributing to waste. This project addresses these issues by helping to reduce food waste.

    Aims

    This project aims to create an automatic shopping list by embedding object detection in a mobile application, allowing users to scan their fridge and detect objects inside it more easily. Missing items are automatically updated and stored in the shopping list, helping users manage their food inventory efficiently.

    About the App

    The application allows users to sign up and create an account to store their fridge inventory. The main screen provides options to view their list, add items, or check the shopping list. The app uses an object detection algorithm integrated with the camera to detect objects in the fridge, which can be added to the list. Users can update their inventory and automatically generate a shopping list for missing items. The data is stored securely in a cloud database (Firebase).

    Takeaway

    This project was built as part of a deep learning subject with a team of two other members. I focused primarily on the GUI, creating the mobile application in Android Studio using Java. I also integrated machine learning algorithms such as Yolov5, compiled into a TensorFlow model for use in the app. Additionally, I learned to implement Firebase as a cloud database for data storage and retrieval, further enhancing my skills in building machine learning algorithms and mobile applications with integrated object detection and cloud storage.
